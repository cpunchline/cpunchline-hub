# base 基础库

C/C++ 基础数据结构,算法和组件库的实现.

## 📊 数据结构分类

### 动态数据结构
- `darray` - 动态数组 (类似 std::vector)
- `deque` - 双端队列 (类似 std::deque)
- `xstring` - 字符串容器 (类似 std::string)

### 静态数据结构
- `list/safe_list/vlist` - 链表系列
- `lstack` - 链式栈
- `ringbuffer` - 环形缓冲区
- `avl/avl_uid` - AVL平衡二叉树
- `rbtree` - 红黑树
- `heap` - 堆结构

## 🔧 算法模块
- base64 编码/解码
- crc 循环冗余校验
- md5 消息摘要算法
- sha1 安全哈希算法

## ⚙️ 基础组件
- `debug_backtrace` - 调用栈追踪
- `threadpool` - 线程池管理
- `threadqueue` - 线程间生产者-消费者队列
- `timercluster` - 定时器集群
- `objectpool` - 内存/对象池
- `syncctxpool` - 同步实体池

## 📚 核心概念说明

### 自平衡树对比
| 特性 | AVL树 | 红黑树 | B树 | B+树 |
|------|-------|--------|-----|------|
| 平衡严格度 | 严格平衡 | 松散平衡 | 多路平衡 | 多路平衡 |
| 查找性能 | 最优 | 良好 | 良好 | 良好 |
| 插入/删除 | 开销较大 | 开销较小 | 磁盘友好 | 磁盘友好 |
| 适用场景 | 内存查找 | 通用场景 | 磁盘索引 | 范围查询 |

### 堆结构
- **性质**: 完全二叉树,父节点值≤/≥子节点值
- **用途**: 优先级队列实现
- **优势**: 快速获取最值元素 O(1),插入删除 O(log n)
