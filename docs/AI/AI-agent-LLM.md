# AI-agent-LLM

Prompt(提示词)是你给 LLM 的输入,它告诉模型你想要什么,就像给助理下达指令,指令越清晰,结果越好.
我们通过 Prompt(提示词) 来与 LLM 交流.
Prompt 的质量直接决定了回答的质量,写好 Prompt 就像给你的 AI 助手下达清晰的指令.
